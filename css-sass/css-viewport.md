# CSS VIEWPORT

## Summary
Notes on device viewport

## Resources
- [Tricks](https://css-tricks.com/fun-viewport-units/)

## Get Device Pixel Ratio

```javascript
window.devicePixelRatio // default to 1 as a fallback
```

## Height

#### Set Viewport Height
```css
100vh; // 100%
25vh; // 25%
```

## Width

#### Set Viewport Width
```css
100vw; // 100%
25vw; // 25%
```

#### Set Viewport Min Width
```css
10vmin; // 10%
```

#### Set Viewport Max Width
```css
10vmax; // 10%
```
