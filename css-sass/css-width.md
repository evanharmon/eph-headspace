## Auto

- [SO](https://stackoverflow.com/questions/27098675/set-default-width-and-min-width-property)
  Auto and min-width can be set at the same time

```css
.modal_dialog img {
  display: inline-block;
  width: auto;
  min-width: 500px;
  max-width: 800px;
}
```
