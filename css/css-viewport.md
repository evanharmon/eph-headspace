# CSS VIEWPORT
[Tricks](https://css-tricks.com/fun-viewport-units/)

## Height

#### Set Viewport Height
```
100vh; // 100%
25vh; // 25%
```

## Width

#### Set Viewport Width
```
100vw; // 100%
25vw; // 25%
```

#### Set Viewport Min Width
`10vmin; // 10%`


#### Set Viewport Max Width
`10vmax; // 10%`
