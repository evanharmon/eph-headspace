# DOCKER IMAGES

## Rename an Image
`$ docker tag <oldname/id> <newname>`

## Stop Docker Image <hash>
`$ docker stop <hash>`

## Build Image
`$ docker build -t "neovim-docker" .`

## List Docker Images
`$ docker images`

## Download Image
`$ docker pull image-name`

## Delete Docker Image <hash> from Downloaded Images
`$ docker rmi <hash>`
