# GOOGLE CLOUD AUTH CLI

## Auth Service Account
```console
$ gcloud auth activate-service-account \
    my-service@my-project.iam.gserviceaccount.com \
    --key-file ~/.gcloud/mykey.json \
    --project myproject
```

## Auth User Account
```console
$ gcloud auth login
```

## Switch Accounts
```console
$ gcloud config set account myaccount
```
