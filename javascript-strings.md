## Get Size In Bytes

use node.js. Much much easier

```javascript
function getBinarySize(string) {
  return Buffer.byteLength(string, "utf8");
}
getBinarySize("ta-daaaaa");
```

## Escape Double Quotations

```javascript
`email = \\${JSON.stringify(email)}\\`;
```
