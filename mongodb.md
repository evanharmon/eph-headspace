# MONGODB

## SERVER
### Start Mongodb Server
Assumes you've created the root `/data/db` directory
`$ mongod`

### Start Mongo and specify a user /data/db
`$ mongod --dbpath $HOME/data/db`

## CLI
### Start Mongodb Command Line
`$ mongo`

### Create Database
`$ use databasename`

### Show Databases
`$ show dbs`

### Show Collections (Tables)
`$ show collections`
