# PYTHON DICTIONARIES

## Create key / value dictionary
`d = {"name": "daniel", "age": 100}`

## Get value
`d["age"]`

## Retrieve list of items
`d.items()`

## Retrieve just keys
`d.keys()`

## Retrieve just values
`d.values()`

## Update value
`d["name"] = "mike"`

## Check if key exists
`d.has_key("name")`
or
`if c in d`
