# RENAME

## Summary
CLI utility for easy renaming of files

## Resources
[SO](https://stackoverflow.com/questions/33127000/renaming-all-files-in-one-folder-on-mac-os-x-yosemite)

## Rename Files In A Directory
Dry-run
```console
rename -n 's/vtl/vm/g' *
```
For Real
```console
rename 's/vtl/vm/g' *
```
