# PROCFS
- similar to sysfs
- presents info about processes
- presents info about system information
- mounted to /proc
- can be used to interface with kernel
- change parameters on the fly

## View
```console
$ cd /proc && ls
```

## Check Memory
```console
$ cat /proc/meminfo
```

## More Readable Memory Info
```console
$ cat /proc/meminfo
```
